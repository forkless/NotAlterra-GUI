# Sidebar Background Video

The sidebar background video is by Pachon in Motion:

- **Creator**: Pachon Curubeto
- **Channel**: [@Pachon.In.Motion](https://instagram.com/Pachon.In.Motion)
- **Location**: Provincia de Buenos Aires, Argentina

The video file (`Assets/loop.webm`) needs to be sourced separately — it is not included in the repository due to licensing. Replace `loop.webm` with any looping background video of your choice, or remove the video element from the XAML to use a static background.
