# Privacy & Security

## NotAlterra Privacy Statement

NotAlterra makes no network connections. No telemetry. No data leaves your machine.

- Zero networking code in the application
- Windows App SDK opt-in telemetry is not enabled
- Configuration stored in HKCU\Software\NotAlterra registry keys
- Backup archives stored in %USERPROFILE%\NotAlterra\
- Transaction log records timestamps only

## Security

- Self-signed installer certificate
- No automatic updates
- No crash reporter
- Read-only metadata inspection — never modifies .sav files in-place
